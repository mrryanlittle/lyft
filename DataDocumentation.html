Data Guide

Below is a list of the data file types contained within your data download and a description of the information included in those files. If a field is blank in your file, it means that we do not have that data for you. Blanks can be expected for some fields, including some historical fields as you go further back in time. Timestamps are in Coordinated Universal Time (UTC) unless otherwise indicated.
